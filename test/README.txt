Build Instructions for Windows 10 MSYS2 MinGW64:
From this directory, run `mkdir build`, `cd build`, `cmake -G Ninja ..`. Then, run `ninja` to compile, and `bin/quadmath-cpp-test` to run the program.

From this directory, run `mkdir build`, `cd build`, `cmake -G Ninja ..`. Then, run `ninja` to compile, and `bin/quadmath-cpp-test` to run the program.
